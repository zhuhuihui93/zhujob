主要特性

基于EasyUI开发

所有的组件都是完全基于EasyUI和支持其所有功能。

PHP, HTML, CSS

使用一些热门的开源框架，网上也能找到对应教程。

完美支持iPad

所有组件在电脑、平板电脑上运行都很正常。

简单易用

不再限制运行环境，同时您可以通过复制和粘贴来实现您的项目中的所有设计。

字体图标

所有图标都采用Font Awesome字体。

扩展兼容

支持插件扩展，同时兼容其他开源系统插件。
